「SELECT 日付, 費目 FROM 家計簿」の形式で記述する。
上記では家計簿テーブルの日付、費目カラムの内容を表示できる。
[SELECT * FROM 家計簿]とすれば、家計簿テーブルの全カラムを表示できる。

条件の指定
SELECT 日付,費目,出金額;
  FROM 家計簿;
 WHERE 出金額 > 3000
  
のように、WHERE　条件で指定した条件に一致するデータのみを表示できる。

データの追加
INSERT INTO 家計簿;
       VALUES ('2018-02-25', '住居費', '3月の家賃', 0, 85000)
INSERT INTOで追加先のテーブルを、VALUESで追加するデータを指定する

データの編集
UPDATE 家計簿;
     SET 出金額 = 90000;
   WHERE 日付 = '2018-02-25'

データの削除
DELETE FROM 家計簿;
   WHERE 日付 = '2018-02-25'
